# Fruityvice Fruit Lookup

A simple Python tool that fetches fruit information (such as family, ID, sugar, and carbohydrate content) from the Fruityvice REST API.  
This tool can be run from the command line or used as a library in Python.

## Usage
Lookup a fruit in the terminal:
```bash
py fruit.py Strawberry
```
Lookup with machine readable output
```bash
py fruit.py Banana --machine
```
or
```bash
py fruit.py Banana -m
```
